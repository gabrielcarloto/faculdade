Ranking de melhores críticas a web moderna

Cada experiência abaixo foi escolhida por sua capacidade de denunciar, com humor ou ironia, decisões de interface que colocam o interesse do produto acima da clareza e do usuário. Algumas são mais realistas, outras são mais exageradas, mas todas são interessantes.

  1. User Inyerface

    Eleva a frustração do usuário ao extremo: nada é intuitivo.

  2. Bad UX Game — NeverDone

    É um sucessor espiritual do UserInyerface. Em um formato de jogo, você deve passar por um site de compra que parece tentar de tudo para você não conseguir completar a compra.

  3. How I Experience Web Today

    Demonstra de maneira interativa más práticas que realmente vemos em sites hoje em dia, como o aparecimento de vários popups antes de te permitir consumir o conteúdo.

  4. Cookie Consent Speed Run

    Outra sátira em torno de banners de cookies, com botões confusos, você deve negar todos os cookies o mais rápido possível.

  5. Motherfucking Website

    Com um site extremamente simples e funcional, critica a complexidade dos websites atuais.
